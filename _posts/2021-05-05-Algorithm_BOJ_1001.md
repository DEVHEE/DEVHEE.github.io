# **[백준 1001번 / Python] A-B**

## **조건**

시간 제한: 2 초 / 메모리 제한: 128 MB

## **문제**

두 정수 A와 B를 입력받은 다음, A-B를 출력하는 프로그램을 작성하시오.

## **입력**

첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## **출력**

첫째 줄에 A-B를 출력한다.

---

## **예제 입력 1**

```
3 2
```

## **예제 출력 1**

```
1
```

---

## **코드**

```python
a, b = map(int, input().split())
print(a-b)
```
